Offcanvas sidebar menu with a twist

In this codepen dive I'll be taking a look at this cool creation from @devilishalchemist on codepen. Take a look and be sure to not touch the red button! Note that the title might imply this makes use of the HTML canvas element, that is not the case.

See the Pen Offcanvas sidebar menu with a twist by Devilish Alchemist (@devilishalchemist) on CodePen.

So how does it work ?

The source code is available in the codepen editor above if you want to skip my boring explanation and experimentations.

So I started with the obvious, preparing our document and laying down the foundation without any animations yet.

Then I started thinking on how I wanted to apply the techniques used in my own implementation. After looking into the source code of the original pen, it is quite an easy implementation - boiling it down to: Rotating the content container on a button press and pulling in some menu items offscreen from the left.

I decided to have my menu items fly in from the top, and rotate the container in a different way. After designing the layouts without animation I wanted to have my menu items line up with the actual container even tho it is slanted. In order to do that I had to use javascript to calculate a dynamic height for the different menu items. I timeboxed the attempt at 3 hours, I wanted to do something more with the actual menu button, but I ran out of time.
