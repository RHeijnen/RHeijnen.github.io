Color reduction with Kmeans

Published: 3 Apr 2019

During college I had a couple of assignments where I had to apply Kmeans to do something - but never anything interesting, mostly just analysing boring fake excel sheets. So I wanted to apply it to something well, usefull is not the word - entertaining ?

So I dug up some old code, got confused by the dumb way I wrote this in college, added a file input and sprinkled some magic on top to actually make it do what I wanted and voila.

I seem to have used some hard limiters and random neighbouring - it works, but if you want to apply this to your own project I suggest you only use this as an example.

See the Pen Color reduction with K-means. by rene heijnen (@ledgeworth) on CodePen.
