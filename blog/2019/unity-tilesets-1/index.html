Tilesets in unity. part I

Published: 21 Dec 2019

What is a tileset?

A simple tileset usualy exists of a grid of 9 blocks(tiles). They are often 16x16 of 32x32 pixels per block in pixelart. But there are no set rules for the size.

The point of these 9 blocks is that they can be repeated to make any simple shapes

How to make them

Now that you have a basic understanding what a tileset is, how do you go about making them ?

Well there are specific tools you can use to create tilesets, it is important to keep in mind that you want your tiles to seemlessly go together to make one big object that does not look like it consists of smaller parts, well - unless you want to ofcourse!

To help us do that we can use some programs like Aseprite - I am sure photoshop and others work just aswell but my experience is in Aseprite

You can even use symmetry tools to draw symmetrically at the same time!

Try and have fun while creating your tiles, you are gonna be doing it for a while.

How to use them in Unity?

For the tutorial on how to use tilesets in unity I've chosen to use this free asset pack so you can work along with the same assets. Note: If you end up useing this for a commercial project - please consider donating to the author(s).

Unity introduced support for tilesets a while ago, so I am going to assume the version you are useing has it supported.

Open your project, or start a new one - I've decided to start a new one. Make sure you have the assets downloaded - you should have a folder called "1 Tiles" somewhere on your computer, locate it and import them into unity (drag and drop on the assets part)

Now you might notice that the folder comes with 'precut' tiles, you can use them but for learning sake - lets learn how to use the tilesheet - the big image containing every tile. Select it in Unity like so:

Make sure you configure the asset as follows: Change pixels per unit to 32 - this asset pack uses 32x32 pixel tiles.
Change filter mode to point (disabled it)
Change Compression to none (we do not want to compress our images)
Change Sprite mode to multiple

Now we are ready to cut it up in 32 by 32 pixel tiles, open the sprite editor from the assets configuration(inspector)

Open the Slice menu (found at the top of the window) and change type from "Automatic" to "Grid By Cell Size" and set the X and Y width to 32 pixels and press slice - you should now see this

Now apply the slicing by hitting "apply" in the righthand corner (top of the window). You just imported your first tilesheet into unity!

Now to use it, I like to work in a grid based system so hitup GameObject -> 2D Object -> Tilemap. Now you should have a new gameobject named "Grid" with a child named "Tilemap" and we are ready to open the tile pallet. Window -> 2D -> tile pallet.

Once open, in the tile pallet create a new empty pallet, it will ask you to designate a folder to save its assets into, unity will save your tiles there. Now you are ready to drag your tileset asset into the window and it should end up like this

Now we can select the tiles we want and literally draw on the grid inside the scene and creating platforming levels has never been so easy!

There is a lot more to say about the grid system in general and there are a lot of other cool things you can do with tiles in unity, like randomly changing tiles to make things more interesting, adding animations to tiles or adding logic to detect corners, so you do not even have to draw them yourself anymore.
